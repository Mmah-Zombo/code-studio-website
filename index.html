Curriculum Overview

Week 1: Python Basics
  • Introduction to Python
  • Variables and Data Types
  • Print Statements and User Input
  • Basic Math Operations
Week 2: Control Flow
  • If/Else Statements
  • Comparison and Logical Operators
  • For and While Loops
  • Building Simple Programs
Week 3: Data Structures
  • Working with Lists
  • Dictionaries and Key-Value Pairs
  • Strings and Basic Operations
  • Mini-project: To-Do List Manager
Week 4: Functions
  • Defining and Calling Functions
  • Parameters and Return Values
  • Organizing Code with Functions
  • Mini-project: Quiz App
Week 5: FastAPI Basics
  • Introduction to APIs
  • Setting up FastAPI
  • Creating Routes (GET, POST)
  • Returning JSON Responses
Week 6: CRUD with FastAPI
  • Path and Query Parameters
  • Request Body with Pydantic
  • In-Memory CRUD Operations
  • Mini-project: Student Records API
Week 7: PostgreSQL Basics
  • Introduction to Databases
  • Setting up PostgreSQL
  • Basic SQL Commands
  • Connecting FastAPI with PostgreSQL
Week 8: API + Database Integration
  • CRUD Operations with PostgreSQL
  • Linking API Routes with Database
  • Error Handling Basics
  • Mini-project: Blog API
Week 9: HTML & CSS
  • HTML Structure and Tags
  • CSS Basics and Selectors
  • Box Model and Flexbox
  • Mini-project: Portfolio Page
Week 10: Tailwind CSS
  • Setting up Tailwind
  • Utility Classes for Styling
  • Responsive Design with Tailwind
  • Mini-project: Portfolio with Tailwind
Week 11: JavaScript Basics
  • Variables and Data Types
  • Functions and Events
  • DOM Manipulation
  • Mini-project: To-Do List App
Week 12: Fullstack Integration
  • Fetching API Data with JavaScript
  • Submitting Forms to API
  • Building a Fullstack App (FastAPI + Postgres + Tailwind + JS)
  • Final Project Presentation

Frequently Asked Questions

What Does “10 Seats” Mean?
Our course is limited to just 10 students to ensure a personalized and engaging learning experience. Therefore only the first 10 students who sign up will secure a spot. Once these slots are filled, registration will close, so be sure to sign up early to guarantee your place in the course.
What are the prerequisites for this bootcamp?
While no prior coding experience is required, a basic understanding of computer operations and a strong desire to learn are essential.
How much time should I dedicate to the bootcamp?
The bootcamp is intensive and requires high commitment. Expect to spend about 4-6 hours per day on coursework, including lectures, coding exercises, and project work.
What kind of support is available during the bootcamp?
During the bootcamp, you'll have access to videos, articles and books in addition to the online classes, that will help you to learn quickly and boost your skills.
